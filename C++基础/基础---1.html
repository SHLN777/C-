一、

1.常量（记录程序中不可更改的数据）
定义常量的两种方式
1.#define 宏定义    #define 常量名  常量值
2.const    const 数据类型 常量名 = 常量值
2.标识符命名规则（常量、变量）
标识符不能是关键字
标识符只能由字母数字下划线组成
第一个字符必须为字母或下划线
标识符中字母区分大小写
3.sizeof关键字（统计数据类型所占的内存大小）
sizeof(int)

short(2) int(4) long(4) long long(8)
4.类型
实型（浮点型）
1.单精度float(4)、7位有效数字
2.双精度double(8)、15~16位有效数字
默认情况下，输出一个小数，会显示6位有效数字
float f1=3.14f; #不加f，认为是double
double d1=3.14;

科学计数法
float f2 = 3e2; //3*40^2

字符型(占一个字符)
 char ch = 'a';
字符变量将相应的ASCLL码放入到存储单元
(int)a=97
5.转义字符（用于表示一些不能显示出来的ASCLL字符）
常用的：\n  \\  \t
\t:水平制表(占8个位置)

cout<<"hello world\n";
cout<<"\\"<<endl;
cout<<"aaa\tbbbb"<<endl; #aaa\t一共占8个空格
6.字符串（字符要单引号 ' '）
1.C风格字符串
    char 变量名[] = "字符串值"
    char 变量名[n] = "字符串值"
2.C++风格字符串
    string 变量名 = "字符串值"
7.布尔类型 bool（占1个字节大小）
bool flag = true;
cout<<flag; //输出1
