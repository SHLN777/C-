常用算法  
 
概述：
  • 算法主要是由头文件<algorithm> <functional> <numeric>
  • <algotirhm>是所有STL头文件中最大的一个，范围涉及到比较、交换、查找、遍历操作、复制、修改等
  • <numeric>体积很小，只包括几个在序列上面 进行简单数学运算的模板函数
  • <functional>定义了一些模板类。用以声明函数对象

transform（搬运容器到另一个容器中）
transform(iterator beg1, iterator end1, iterator beg2, _func);
源容器：v
目标容器：v1

class Transform
{
public:
    int operator()(int v)   //可以加一些逻辑运算
    {
        return v;
    }
};
v1.resize(v.size());  //目标容器需要提前开辟空间
transform( v.begin(), v.end(), v1.begin(), Transform() );

adjacent_find（查找相邻重复元素）
adjacent_find(iterator beg, iterator end);
//返回相邻元素的第一个位置的迭代器
vector<int>v;

vector<int>::iterator pos = adjacent_find(v.begin(),v.end());
if (pos == v.end())
{
    cout<<"没有找到！";
}
else
{
    cout<<"找到！";
}


binary_search（查找指定元素是否存在）
bool binary_search(iterator beg, iterator end, value);

vector<int>v;

bool ret=binary_search(v.begin(),v.end(),9);

{
    cout<<"找到！";

else
